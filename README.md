# OCD 256x8 SRAM test on Global Foundaries 180 nm

Test design for the [ocd 256x8 SRAM](https://github.com/RTimothyEdwards/gf180mcu_ocd_ip_sram) going out on the Tiny Tapeout
`26f0p3` experimental shuttle using the Global Foundaries 180 nm node.
